# vue-cli-study

> 这是一个vue-project项目

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
##修改首页tabBar图标
1.找到mui的扩展图标库
2.将扩展图标拷贝到项目中
3.在main.js中引入扩展图标的样式
4.使用扩展图标
